Integration Examples

Basic Setup

// Initialize 2FA for a user
const setupData = await TwoFactorAuth.generateSecret();
console.log('QR Code URL:', setupData.qrCodeUrl);
console.log('Manual Entry:', setupData.manualEntryKey);

Verify User Setup

// Verify and enable 2FA
const result = await TwoFactorAuth.enable2FA('123456');
if (result.success) {
  console.log('2FA enabled!', result.backupCodes);
}

Verify Login Code

// Verify 2FA code during login
const verification = await TwoFactorAuth.verify2FACode('123456');
if (verification.success) {
  // Allow user to proceed
}

Documentation

Important Notes
  • 2FA secrets are encrypted before storing in the database
  • Backup codes are single-use and automatically marked as used
  • The system supports a 1-step time window tolerance for clock drift
  • All 2FA activities are logged for security auditing
Security Considerations
  • Always use HTTPS in production environments
  • Store 2FA secrets securely with proper encryption
  • Implement rate limiting for 2FA verification attempts
  • Provide clear backup and recovery procedures
